Ces informations concernent par exemple le numéro de version d'un logiciel, les informations de service, la conception de l'infrastructure, les concepts détaillés ou le code source d'un logiciel/application. Lorsqu'il s'agit de mécanismes électroniques, les schémas de circuit, les configurations de cartes, les mappages ou les schémas peuvent contribuer à l'obtention de résultats de haute qualité. La collaboration nécessite une procédure de test active contre les infrastructures de l'entreprise à tester.
